# 🍏 Macro-Logic PWA

**Macro-Logic** es una aplicación web progresiva (PWA) diseñada para el cálculo de macronutrientes y gestión de equivalencias de alimentos. Enfocada en la velocidad y la utilidad para sustituciones inteligentes.

> **Estado:** Versión 1.0 (MVP Estático) Completada ✅

---

## 🛠️ Stack Tecnológico

- **Framework:** Next.js (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS v4
- **Componentes:** Shadcn/UI (Radix + Nova)
- **Tipografía:** Quicksand

---

## ✨ Funcionalidades v1.0

- **Buscador Dinámico:** Filtrado en tiempo real de alimentos.
- **Calculadora de Macros:** Cálculo instantáneo basado en gramos introducidos.
- **Rutas Dinámicas:** Navegación fluida entre lista y detalle.
- **Mobile First:** Optimizado para uso en dispositivos móviles.

---

## 🚀 Instalación

1. Clonar: `git clone <repo-url>`
2. Instalar: `npm install`
3. Desarrollo: `npx next dev -H 0.0.0.0`
